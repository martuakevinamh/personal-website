# Experience Images

Place your experience/activity photos here.

## File Naming Convention:

- Organization: `org-[id]-[number].jpg`
- Committee: `com-[id]-[number].jpg`

## Examples:

- org-1-1.jpg (Organization 1, Image 1)
- org-1-2.jpg (Organization 1, Image 2)
- com-1-1.jpg (Committee 1, Image 1)
